## Install

```bash
npm i @blacktiger/tmdb
```

## Usage

The API key is validated when the instance is created.

```ts
import TMDB from '@blacktiger/tmdb';

const tmdb = new TMDB(/* Your API Key */, /* Optional Language, Default en-US */);
```
